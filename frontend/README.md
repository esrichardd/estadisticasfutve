# Frontend - Estadisticas FUTVE

Aplicacion web de Estadisticas FUTVE.

## Inicio rapido

```bash
pnpm install
pnpm dev
```

Abrir en el navegador:

```txt
http://localhost:3000/es
```

## Scripts

```bash
pnpm dev      # servidor de desarrollo
pnpm build    # build de produccion
pnpm start    # servir build de produccion
pnpm lint     # ejecutar ESLint
```

## Convenciones de trabajo

- Usa `pnpm` para instalar dependencias y ejecutar scripts.
- Las rutas publicas usan prefijo de idioma, por ejemplo `/es` y `/en`.
- Para decisiones de estructura, capas o dependencias internas, consulta la documentacion de arquitectura.
